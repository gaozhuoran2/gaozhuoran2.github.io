### 成本函数
对于每一个 w,b 生成的$f_{w,b}=wx+b$我们用成本函数检验拟合函数的优秀程度

$$
J(w,b)=\frac{1}{2m}\sum_{i=0}^{m-1}(f(x_i)-y_i)^2
$$

分别对 w,b 求导得：

$$
\frac{\partial J(w,b)}{\partial w}=\frac{1}{m}\sum_{i=0}^{m-1}(f(x_i)-y_i)x_i
$$

$$
\frac{\partial J(w,b)}{\partial b}=\frac{1}{m}\sum_{i=0}^{m-1}(f(x_i)-y_i)
$$

### 梯度下降
成本函数要求越小越好，显然，延切线方向更新 w,b 即可

$$
w=w-\alpha \frac{\partial J(w,b)}{\partial w}
$$

$$
b=b-\alpha \frac{\partial J(w,b)}{\partial b}
$$

即得最优解

### 多元梯度下降
与一元梯度下降类似

$$
\frac{\partial J(\boldsymbol{w},b)}{\partial w_j}
= \frac{1}{m}\sum_{i=0}^{m-1}\big(f_{\boldsymbol{w},b}(\boldsymbol{x_i}) - y_i\big)x_i^{(j)}
$$

$$
\frac{\partial J(\boldsymbol{w},b)}{\partial b}
= \frac{1}{m}\sum_{i=0}^{m-1}\big(f_{\boldsymbol{w},b}(\boldsymbol{x_i}) - y_i\big)
$$

$$
w_j = w_j - \alpha \frac{\partial J(\boldsymbol{w},b)}{\partial w_j}
$$

$$
b = b - \alpha \frac{\partial J(\boldsymbol{w},b)}{\partial b}
$$

### 特征缩放

对于每个样例，他的不同特征的数值范围可能差别很大，会影响梯度下降的速度，所以我们进行特征缩放，让每个特征的值都大约分布在 -1 和 1 之间

平均归一化：

$$
x_i^{(j)} = \frac{x_i^{(j)} - \mu_j}{\max - \min}
$$

z-score归一化：

$$
x_i^{(j)} := \frac{x_i^{(j)} - \mu_j}{\sigma_j}
$$

 ### 直接计算 w,b

 利用 $\boldsymbol{w} = \left(\boldsymbol{X}^\mathrm{T} \boldsymbol{X}\right)^{-1} \boldsymbol{X}^\mathrm{T} \boldsymbol{y}$直接求解

```python
import numpy as np
from sklearn.preprocessing import StandardScaler
from sklearn.linear_model import LinearRegression
np.set_printoptions(precision=2)

X_train = np.array([[1,23],[2,34],[3,56],[4,77],[5,99]])
y_train = np.array([2,4,6,8,10])

scaler = StandardScaler()
X_norm=scaler.fit_transform(X_train)

model = LinearRegression()

#直接求 w,b
model.fit(X_norm, y_train)
print("权重 w =", model.coef_)
print("截距 b =", model.intercept_)

#预测新的
X_test = np.array([[6,113],[7,134]])
y_pred = model.predict(X_test)
print("预测结果：", y_pred)
```

```python
import numpy as np
from sklearn.preprocessing import StandardScaler
from sklearn.linear_model import SGDRegressor
np.set_printoptions(precision=2)

X_train = np.array([[1,23],[2,34],[3,56],[4,77],[5,99]])
y_train = np.array([2,4,6,8,10])

scaler = StandardScaler()
X_norm=scaler.fit_transform(X_train)

sgd=SGDRegressor(max_iter=10000)

sgd.fit(X_norm,y_train)
#注意：要传 X_norm 不是 X_train
print("w =", sgd.coef_)
print("b =", sgd.intercept_)

X_test = np.array([[6,113],[7,134]])
X_test_norm = scaler.transform(X_test)
y_pred = sgd.predict(X_test_norm)
print(y_pred)
```
